# Judul Materi Belajar JavaScript: Dasar sampai Mahir

Daftar judul materi ini disusun berdasarkan referensi pada bagian **"Tutorial JavaScript untuk Pemula"** di [sumber_materi.md](sumber_materi.md), serta kurikulum standar dari JavaScript.info, MDN Learn JavaScript, W3Schools JavaScript, Eloquent JavaScript, dan FreeCodeCamp. Urutan dari dasar sampai mahir mengikuti alur: pengenalan → sintaks & tipe data → kontrol alur → fungsi → array & objek → DOM & event → form & validasi → ES6+ & async → topik lanjut.

**Pemetaan ke [cpl_cpmk.md](cpl_cpmk.md):** Materi ini mendukung **Sub-CPMK 2.3** (DOM, event handling, validasi form) dan **Sub-CPMK 2.4** (library/framework front-end; judul eksplisit ditambah di Tingkat Lanjut). Semua judul sesuai dengan CPMK2.

---

## Tingkat Dasar

1. Pengenalan JavaScript dan perannya di web (client-side)
2. Memulai: menjalankan JavaScript di browser dan konsol
3. Sintaks dasar: statement, komentar, dan aturan penulisan
4. Variabel: var, let, dan const
5. Tipe data: number, string, boolean, undefined, null
6. Operator: aritmetika, perbandingan, logika, dan penugasan
7. Konversi tipe dan typeof
8. String: literal, escape, dan metode dasar
9. Percabangan: if, else if, else, dan switch
10. Perulangan: for, while, do...while, break, continue
11. Fungsi: deklarasi, parameter, dan return
12. Scope: global, function, dan block (let/const)
13. Array: membuat, mengakses indeks, dan metode dasar (push, pop, length)
14. Objek: literal, properti, dan metode
15. Pengenalan DOM (Document Object Model)
16. Mengakses elemen DOM dan mengubah konten
17. Event dasar: click, load, submit

---

## Tingkat Menengah

1. Fungsi lanjutan: expression, arrow function, default parameter, rest
2. Array lanjutan: map, filter, reduce, forEach, find, sort
3. Objek lanjutan: this, constructor, dan iterasi
4. String lanjutan dan template literal
5. Number, Math, dan pembulatan
6. Date dan penanganan waktu
7. JSON: parse dan stringify
8. Penanganan error: try, catch, throw
9. Strict mode dan praktik penulisan yang baik
10. DOM: traversing, membuat node, dan menghapus elemen
11. Event: addEventListener, objek event, dan propagation
12. Form: mengambil nilai input dan validasi di client
13. Timer: setTimeout dan setInterval
14. Pengenalan asynchronous: callback
15. ES6+: destructuring, spread operator, dan modul (import/export)
16. Class dan pewarisan (ES6)
17. Regular expression (RegExp) dasar

---

## Tingkat Lanjut / Mahir

1. Promise dan async/await
2. Fetch API dan permintaan HTTP dari browser
3. Closure dan lexical scope
4. Prototype dan pewarisan (tinjauan mendalam)
5. Pola desain dasar: module pattern, singleton
6. Debugging dan alat pengembang (DevTools)
7. Pengujian dasar (konsep unit test)
8. Performa: praktik baik dan profiling
9. Keamanan: XSS dan sanitasi input
10. **Menggunakan library atau framework front-end (Bootstrap, jQuery, atau Vue/React) untuk antarmuka dinamis** *(mendukung Sub-CPMK 2.4)*
11. Alat modern: pengenalan bundler dan tooling (opsional)
12. Praktik proyek: aplikasi interaktif atau integrasi dengan backend
