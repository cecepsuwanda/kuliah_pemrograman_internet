# CPL & CPMK — Pemrograman Web

## CPL (Capaian Pembelajaran Lulusan)

**CPL04**
Kemampuan mendesain, mengimplementasi dan mengevaluasi solusi berbasis computing yang memenuhi kebutuhan-kebutuhan computing pada sebuah disiplin program.

**CPL05**
Mengidentifikasi dan menganalisis kebutuhan-kebutuhan pengguna dan mempertimbangkannya dalam memilih, membuat, mengintegrasi, mengevaluasi, dan mengadministrasi sistem berbasis computing.

---

## CPMK (Capaian Pembelajaran Mata Kuliah)

**CPMK1 — Analisis Kebutuhan Pengguna dan Perancangan Antarmuka** *(Turunan CPL05)*  
Mahasiswa mampu mengidentifikasi kebutuhan fungsional dan non-fungsional pengguna untuk merancang solusi web yang intuitif melalui pembuatan wireframe dan mockup (UI/UX) yang mempertimbangkan aspek aksesibilitas dan pengalaman pengguna.

**CPMK2 — Implementasi Solusi Web Front-End** *(Turunan CPL04)*  
Mahasiswa mampu mendesain dan mengimplementasikan antarmuka web yang responsif dan dinamis menggunakan teknologi standar (seperti HTML5, CSS3, dan JavaScript) serta framework modern untuk memenuhi spesifikasi perangkat pengguna yang beragam.

**CPMK3 — Pengembangan Sistem Back-End dan Integrasi Data** *(Turunan CPL04 dan CPL05)*  
Mahasiswa mampu membangun arsitektur sisi server (back-end), mengelola basis data, dan mengintegrasikan API (Application Programming Interface) untuk menciptakan sistem berbasis computing yang fungsional dan aman sesuai dengan disiplin program.

**CPMK4 — Evaluasi, Deployment dan Administrasi Web** *(Turunan CPL04 dan CPL05)*  
Mahasiswa mampu melakukan pengujian (testing), mengevaluasi keamanan serta performa aplikasi web, dan melakukan proses deployment ke server produksi (administrasi sistem) untuk memastikan aplikasi dapat diakses secara optimal oleh publik.

---

## Sub-CPMK (Indikator Pencapaian)

Sub-CPMK berikut disusun sebagai indikator terukur yang dapat diuji melalui tugas, kuis, atau praktikum. Penyusunan mengacu pada [sumber_materi.md](sumber_materi.md) dan referensi pemrograman web terbuka (MDN, web.dev, PHP The Right Way, dan kurikulum OBE pemrograman web).

### Turunan CPMK1 — Analisis Kebutuhan Pengguna dan Perancangan Antarmuka

- **Sub-CPMK 1.1:** Mengidentifikasi kebutuhan fungsional dan non-fungsional pengguna untuk solusi web
- **Sub-CPMK 1.2:** Membuat wireframe dan mockup antarmuka web yang mendukung alur pengguna (user flow)
- **Sub-CPMK 1.3:** Menerapkan prinsip aksesibilitas (a11y) dan pengalaman pengguna (UX) dalam desain antarmuka

### Turunan CPMK2 — Implementasi Solusi Web Front-End

- **Sub-CPMK 2.1:** Membuat halaman web dengan HTML5 (struktur, elemen semantik, form, media)
- **Sub-CPMK 2.2:** Menerapkan CSS3 untuk styling, layout (Flexbox/Grid), dan desain responsif
- **Sub-CPMK 2.3:** Mengimplementasikan interaktivitas dengan JavaScript (DOM, event handling, validasi form)
- **Sub-CPMK 2.4:** Menggunakan library atau framework front-end (mis. Bootstrap, jQuery, atau Vue/React) untuk antarmuka dinamis

### Turunan CPMK3 — Pengembangan Sistem Back-End dan Integrasi Data

- **Sub-CPMK 3.1:** Mengimplementasikan logika server-side dengan PHP (pemrosesan form, struktur dasar, OOP)
- **Sub-CPMK 3.2:** Mengelola basis data dan mengimplementasikan operasi CRUD dengan MySQL serta koneksi dari PHP
- **Sub-CPMK 3.3:** Mengintegrasikan API dan menghubungkan front-end dengan back-end (request/response, data terstruktur)

### Turunan CPMK4 — Evaluasi, Deployment dan Administrasi Web

- **Sub-CPMK 4.1:** Melakukan pengujian (testing) dan evaluasi performa aplikasi web
- **Sub-CPMK 4.2:** Menerapkan prinsip keamanan web: validasi input, prepared statement, session, mitigasi XSS dan CSRF
- **Sub-CPMK 4.3:** Melakukan deployment ke lingkungan produksi dan administrasi dasar (server/web, basis data)
