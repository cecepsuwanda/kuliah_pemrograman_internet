# Judul Materi Belajar HTML: Dasar sampai Mahir

Daftar judul materi ini disusun berdasarkan referensi pada bagian **"Tutorial HTML untuk Pemula"** di [sumber_materi.md](sumber_materi.md), serta kurikulum standar dari MDN Learn HTML, web.dev Learn HTML, dan W3Schools HTML Tutorial. Urutan dari dasar sampai mahir mengikuti alur: pengenalan → sintaks & struktur → teks & link → gambar → tabel → form → semantik → media → aksesibilitas → validasi & standar.

**Pemetaan ke [cpl_cpmk.md](cpl_cpmk.md):** Materi ini mendukung **Sub-CPMK 2.1** (struktur, elemen semantik, form, media) dan **Sub-CPMK 1.3** (aksesibilitas dalam desain antarmuka). Semua judul di bawah sesuai dengan CPMK2 dan CPMK1; tidak ada yang ditandai tidak sesuai.

---

## Tingkat Dasar

1. Pengenalan HTML dan perannya di web
2. Memulai: editor, browser, dan file HTML
3. Sintaks dasar HTML: tag, elemen, dan atribut
4. Struktur dokumen HTML5 (doctype, html, head, body)
5. Bagian head dan metadata halaman
6. Heading dan paragraf
7. Teks: emphasis, importance, dan format dasar
8. Daftar: unordered, ordered, dan description list
9. Fitur teks lanjutan: kutipan, kode, subscript, superscript
10. Membuat link (hyperlink) dan praktik terbaik
11. Gambar: elemen img dan figure

---

## Tingkat Menengah

1. Tabel HTML: dasar (baris, sel, header)
2. Tabel: pengelompokan, scope, dan aksesibilitas
3. Form HTML: elemen form, input, label, dan button
4. Form: tipe input, select, textarea, dan validasi dasar
5. Elemen semantik HTML5: header, nav, main, section, article, aside, footer
6. Merancang struktur halaman dengan elemen semantik
7. Media: audio dan video
8. Gambar vektor (SVG) dalam HTML
9. Embedding: iframe, object, dan embed

---

## Tingkat Lanjut / Mahir

1. Aksesibilitas (a11y): prinsip dan praktik di HTML
2. Aksesibilitas tabel dan form
3. Validasi HTML dan alat validator
4. Debugging HTML: menemukan dan memperbaiki error
5. Best practices HTML dan standar (WHATWG / W3C)
