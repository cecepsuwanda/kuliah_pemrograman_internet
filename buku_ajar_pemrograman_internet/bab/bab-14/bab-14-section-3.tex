\section{Pengenalan Framework dan Deployment}

Framework front-end (React, Vue) dan back-end (Laravel, Express) mempercepat pengembangan dengan struktur dan fitur siap pakai \cite{react}, \cite{vuejs}, \cite{express}. Pengenalan framework dapat dilakukan setelah penguasaan dasar HTML, CSS, JavaScript, dan server-side \cite{mdn-learn}. Deployment berarti memindahkan aplikasi ke server produksi; layanan seperti Vercel, Netlify, atau shared hosting dapat digunakan \cite{github}. Praktik baik: gunakan Git untuk version control, dokumentasikan langkah deploy \cite{git}.
