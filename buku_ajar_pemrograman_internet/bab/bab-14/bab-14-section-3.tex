\section{Pengenalan Framework dan Deployment}

Framework front-end (React, Vue) dan back-end berbasis PHP (mis. Laravel) mempercepat pengembangan dengan struktur dan fitur siap pakai \cite{react}, \cite{vuejs}. Pengenalan framework dapat dilakukan setelah penguasaan dasar HTML, CSS, JavaScript, dan server-side \cite{mdn-learn}. Deployment berarti memindahkan aplikasi ke server produksi; layanan seperti Vercel, Netlify, atau shared hosting dapat digunakan \cite{github}. Praktik baik: gunakan Git untuk version control, dokumentasikan langkah deploy \cite{git}.
