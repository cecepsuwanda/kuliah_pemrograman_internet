\section{Sintaks, Tipe Data, dan Kontrol Alur JavaScript}

JavaScript adalah bahasa pemrograman yang dijalankan di browser untuk menambahkan interaktivitas pada halaman web \cite{mdn-js}. JavaScript mendukung tipe data primitif (number, string, boolean, undefined, null) dan tipe referensi (object, array, function) \cite{js-info}. Variabel dideklarasikan dengan \texttt{let}, \texttt{const}, atau \texttt{var}; \texttt{const} untuk nilai yang tidak diubah, \texttt{let} untuk nilai yang dapat berubah \cite{eloquent-js}. Kontrol alur meliputi \texttt{if/else}, \texttt{switch}, \texttt{for}, \texttt{while}, dan \texttt{forEach} untuk iterasi array \cite{mdn-learn-js}. Fungsi didefinisikan dengan \texttt{function} atau arrow function \texttt{() => \{ \}} \cite{js-info}.
