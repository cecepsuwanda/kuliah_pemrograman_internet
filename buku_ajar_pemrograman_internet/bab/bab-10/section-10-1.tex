\section{Koneksi Basis Data dan Query}

Aplikasi web sering memerlukan penyimpanan data persisten; basis data relasional seperti MySQL umum digunakan \cite{mysql}. Koneksi ke MySQL dari PHP dapat menggunakan PDO atau mysqli \cite{php-manual}. Query SQL (SELECT, INSERT, UPDATE, DELETE) dijalankan melalui API koneksi; hasil query ditampilkan di halaman atau dikembalikan sebagai JSON \cite{w3schools-sql}. Penting untuk menggunakan prepared statement atau parameterisasi query guna mencegah SQL injection \cite{owasp-sql}. Dokumentasi MySQL menjelaskan sintaks SQL dan optimasi \cite{mysql}.
