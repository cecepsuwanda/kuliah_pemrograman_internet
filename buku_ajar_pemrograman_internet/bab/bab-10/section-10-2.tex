\section{Operasi CRUD Berbasis Web}

CRUD (Create, Read, Update, Delete) adalah operasi dasar untuk data. Create: form POST mengirim data, server INSERT ke tabel. Read: GET request, server SELECT dan menampilkan data. Update: form POST dengan ID, server UPDATE. Delete: link atau tombol dengan konfirmasi, server DELETE \cite{php-manual}. Implementasi CRUD menghubungkan front-end (form, tabel) dengan back-end (script PHP yang menjalankan query) \cite{mysql}. Setiap operasi harus memvalidasi input dan menggunakan prepared statement \cite{owasp-sql}, \cite{owasp-input}.
