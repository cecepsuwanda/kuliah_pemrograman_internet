\section{Konsep REST API dan Format JSON}

REST (Representational State Transfer) adalah gaya arsitektur untuk API web \cite{rest-tutorial}. Endpoint REST menggunakan URL dan metode HTTP (GET, POST, PUT, DELETE) untuk operasi; respons biasanya dalam format JSON \cite{owasp-rest}. JSON (JavaScript Object Notation) adalah format pertukaran data yang ringan dan mudah dibaca \cite{json-org}. Struktur JSON terdiri dari object (\texttt{\{\}}) dan array (\texttt{[]}); nilai dapat berupa string, number, boolean, null, object, atau array \cite{mdn-api}. Standar REST menggunakan status code HTTP untuk mengomunikasikan hasil \cite{http-status}.
