\section{Flexbox dan CSS Grid}

Flexbox dan CSS Grid adalah sistem layout modern untuk CSS \cite{mdn-css}. Flexbox dirancang untuk layout satu dimensi (baris atau kolom) dan sangat cocok untuk komponen seperti navigasi, card, atau baris tombol \cite{csstricks}. Properti utama Flexbox: \texttt{display: flex}, \texttt{flex-direction}, \texttt{justify-content}, \texttt{align-items}, \texttt{flex-wrap} \cite{webdev}. Flexbox memudahkan perataan dan distribusi ruang antar item tanpa float atau positioning yang rumit.

CSS Grid dirancang untuk layout dua dimensi (baris dan kolom sekaligus) \cite{mdn-css}. Properti \texttt{display: grid}, \texttt{grid-template-columns}, \texttt{grid-template-rows}, dan \texttt{gap} memungkinkan pembuatan layout kompleks dengan kode yang ringkas \cite{w3c-css}. Grid ideal untuk halaman dengan area header, sidebar, main content, dan footer. Browser modern mendukung Flexbox dan Grid; kompatibilitas dapat dicek di \cite{caniuse}.
