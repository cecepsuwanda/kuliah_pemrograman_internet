\section{Desain Responsif}

Desain responsif membuat halaman web tampil baik di berbagai ukuran layar (desktop, tablet, ponsel) \cite{mdn-learn-css}. Teknik utama: viewport meta tag (\texttt{<meta name="viewport" content="width=device-width">}), media queries (\texttt{@media (max-width: 768px) \{ ... \}}), unit relatif (\texttt{rem}, \texttt{\%}, \texttt{vw}), dan layout yang fleksibel (Flexbox, Grid) \cite{webdev}. Media query memungkinkan penerapan aturan CSS berbeda berdasarkan lebar layar \cite{csstricks}. Pendekatan mobile-first berarti merancang untuk layar kecil terlebih dahulu, lalu menambahkan aturan untuk layar lebih besar \cite{bootstrap}.
