\section{Event Handling pada Form}

Event handling memungkinkan JavaScript merespons aksi pengguna (klik, submit form, input, dll.) \cite{mdn-js}. Untuk form, event \texttt{submit} terpicu saat form dikirim; kita dapat mencegah pengiriman default dengan \texttt{event.preventDefault()} untuk validasi atau pemrosesan AJAX \cite{js-info}. Event \texttt{input} atau \texttt{change} berguna untuk validasi real-time saat pengguna mengetik \cite{eloquent-js}. Menggunakan \texttt{addEventListener} lebih disarankan daripada atribut inline \texttt{onclick} karena memisahkan perilaku dari markup dan mendukung multiple listener \cite{mdn-learn-js}.
