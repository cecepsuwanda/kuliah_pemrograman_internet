\begin{aktivitas}
  \item Instalasi MySQL (via XAMPP); praktik akses CLI dan phpMyAdmin. Buat database dan tabel dengan berbagai tipe data.
  \item Praktik DDL: ALTER TABLE. Praktik DML: SELECT dengan WHERE, operator, ORDER BY, LIMIT; INSERT, UPDATE, DELETE.
  \item Gunakan fungsi agregat (COUNT, SUM, AVG, MIN, MAX), GROUP BY, HAVING, DISTINCT, alias. Tangani nilai NULL.
  \item Desain relasi tabel: primary key, foreign key, normalisasi (1NF--3NF). Praktik JOIN (INNER, LEFT, RIGHT) dan subquery.
  \item Buat index dan analisis performa query dengan EXPLAIN. Praktik backup/restore dengan mysqldump.
  \item Koneksi PHP--MySQL dengan MySQLi (OOP). Praktik fetch: \code{fetch\_assoc()}, \code{fetch\_all()}. Bangun aplikasi CRUD lengkap.
  \item Gunakan prepared statement (MySQLi dan PDO) untuk mencegah SQL injection. Bandingkan MySQLi vs PDO.
  \item Praktik MySQL lanjut: VIEW, stored procedure, trigger, transaksi (BEGIN/COMMIT/ROLLBACK).
  \item Kelola user dan hak akses MySQL. Rancang skema database lengkap dengan diagram ER.
  \item Terapkan keamanan web: validasi input, sanitasi output, mitigasi XSS dan CSRF dengan token.
  \item Pelajari exception, namespace, Composer, dan struktur proyek PHP sesuai PSR.
  \item Bangun REST API sederhana dengan PHP; hubungkan front-end (Fetch API) dengan back-end.
  \item Tulis unit test dengan PHPUnit. Siapkan checklist deployment untuk produksi.
\end{aktivitas}

\begin{latihan}
  \item Jelaskan perbedaan DDL dan DML; berikan contoh minimal 2 perintah masing-masing.
  \item Apa peran primary key dan foreign key dalam menjaga integritas data?
  \item Tulis query dengan JOIN untuk menggabungkan data dari dua tabel; jelaskan perbedaan INNER, LEFT, dan RIGHT JOIN.
  \item Apa itu normalisasi? Jelaskan 1NF, 2NF, dan 3NF dengan contoh.
  \item Jelaskan bagaimana prepared statement mencegah SQL injection dibandingkan penggabungan string.
  \item Bandingkan MySQLi dan PDO dari sisi portabilitas, fitur, dan gaya API.
  \item Buat endpoint REST API sederhana yang mengembalikan data produk dalam format JSON.
  \item Apa itu CSRF token dan mengapa diperlukan di setiap form? Jelaskan mekanismenya.
\end{latihan}

\begin{asesmen}
Asesmen berupa proyek: \textbf{Aplikasi Web CRUD Full-Stack} dengan komponen:
\begin{itemize}
  \item Desain database ternormalisasi (minimal 3 tabel dengan relasi FK) dan diagram ER.
  \item Implementasi CRUD dengan PHP dan MySQLi/PDO menggunakan prepared statement.
  \item REST API minimal 2 endpoint (GET dan POST) dengan respons JSON.
  \item Integrasi front-end menggunakan JavaScript Fetch API.
  \item Keamanan: validasi input, sanitasi output, CSRF token, password hashing.
  \item (Opsional) Pengujian unit dengan PHPUnit; deployment checklist.
\end{itemize}
Penilaian: desain database (20\%), implementasi CRUD (25\%), API (15\%), keamanan (25\%), dokumentasi (15\%).
\end{asesmen}

\begin{checklist}
  \item Saya memahami database relasional, SQL, DDL (CREATE, ALTER, DROP), dan DML (SELECT, INSERT, UPDATE, DELETE).
  \item Saya dapat menggunakan WHERE, operator, ORDER BY, LIMIT, fungsi agregat, GROUP BY, HAVING, DISTINCT, alias, dan NULL.
  \item Saya memahami relasi tabel, primary key, foreign key, JOIN (INNER/LEFT/RIGHT), subquery, dan normalisasi (1NF--3NF).
  \item Saya dapat membuat dan menggunakan index; menganalisis query dengan EXPLAIN; melakukan backup/restore.
  \item Saya dapat membuat koneksi MySQLi (OOP), menjalankan query, dan menggunakan metode fetch.
  \item Saya mampu membangun aplikasi CRUD dengan prepared statement dan memahami pencegahan SQL injection.
  \item Saya dapat menggunakan PDO dan memahami perbedaannya dengan MySQLi.
  \item Saya mengenal VIEW, stored procedure, trigger, transaksi, dan keamanan user MySQL.
  \item Saya mampu menerapkan validasi input, sanitasi output, dan mitigasi XSS/CSRF.
  \item Saya memahami exception, namespace, Composer, dan struktur proyek PHP (PSR).
  \item Saya dapat membangun REST API sederhana dan menghubungkan front-end (Fetch API) dengan back-end.
  \item Saya mengetahui dasar pengujian (PHPUnit) dan checklist deployment produksi.
\end{checklist}

\begin{rangkuman}
Bab ini membahas MySQL dan operasi CRUD secara komprehensif dalam empat area utama. \textbf{MySQL Dasar} (section 1--14) mencakup pengenalan database dan SQL, instalasi MySQL, akses CLI dan GUI, DDL (CREATE DATABASE, CREATE TABLE, ALTER TABLE), DML (SELECT, INSERT, UPDATE, DELETE), operator, ORDER BY/LIMIT, fungsi agregat, GROUP BY/HAVING, DISTINCT/alias, serta penanganan NULL.

\textbf{Relasi dan Lanjut} (section 15--22) membahas relasi tabel dan konsep kunci (PK/FK), JOIN (INNER, LEFT, RIGHT), subquery, PRIMARY KEY vs UNIQUE, FOREIGN KEY dan referential integrity, normalisasi (1NF/2NF/3NF), index, serta backup dan restore dengan mysqldump.

\textbf{Integrasi PHP--MySQL} (section 23--27) mengajarkan koneksi MySQLi (OOP), metode fetch, CRUD aplikasi web dengan prepared statement, pencegahan SQL injection, serta PDO sebagai alternatif yang portable.

\textbf{MySQL Advanced dan PHP Lanjut} (section 28--46) meliputi VIEW, stored procedure, trigger, transaksi (ACID), optimasi EXPLAIN, keamanan user, perancangan database, keamanan web (validasi/sanitasi, XSS, CSRF), exception, namespace, Composer, struktur proyek, PHP CLI, REST API, integrasi front-back dengan Fetch API, pengujian PHPUnit, dan deployment.

\textbf{Poin Kunci:}
\begin{itemize}
  \item SQL DDL/DML, relasi, JOIN, normalisasi, dan index adalah fondasi pengelolaan basis data.
  \item Prepared statement (MySQLi/PDO) wajib untuk keamanan; jangan pernah gabungkan input ke query.
  \item REST API dengan JSON menjadi standar komunikasi front-end dan back-end modern.
  \item Keamanan, pengujian, dan deployment yang baik melengkapi siklus pengembangan full-stack.
\end{itemize}
\end{rangkuman}
