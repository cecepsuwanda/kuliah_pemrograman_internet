\begin{aktivitas}
  \item Siapkan proyek mini dan lakukan deployment sederhana.
  \item Buat catatan konfigurasi server dan database.
  \item Presentasikan hasil dan minta umpan balik.
\end{aktivitas}

\begin{latihan}
  \item Jelaskan dua perbedaan utama antara lokal dan produksi.
  \item Buat checklist langkah deployment sederhana.
\end{latihan}

\begin{asesmen}
Asesmen akhir berupa proyek terintegrasi dan presentasi hasil kerja.
\end{asesmen}

\begin{checklist}
  \item Saya memahami konsep deployment dasar.
  \item Saya tahu langkah konfigurasi produksi.
  \item Saya mampu mengevaluasi hasil belajar sendiri.
\end{checklist}

\begin{rangkuman}
Bab ini menutup pembahasan dengan deployment, administrasi dasar, dan evaluasi akhir. Anda diajak mengintegrasikan semua keterampilan yang telah dipelajari dalam satu proyek.

\textbf{Poin Kunci:}
\begin{itemize}
  \item Deployment memerlukan konfigurasi dan keamanan yang lebih ketat.
  \item Administrasi dasar mencakup server dan database.
  \item Evaluasi akhir membantu mengukur pencapaian kompetensi.
\end{itemize}
\end{rangkuman}
