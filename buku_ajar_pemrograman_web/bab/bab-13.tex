\begin{aktivitas}
  \item Buat endpoint sederhana yang mengembalikan JSON.
  \item Tampilkan data JSON di halaman web.
  \item Lakukan pengujian fungsional pada form yang dibuat.
\end{aktivitas}

\begin{latihan}
  \item Jelaskan mengapa JSON banyak digunakan pada API.
  \item Sebutkan dua cara mempercepat performa aplikasi web.
\end{latihan}

\begin{asesmen}
Asesmen berupa tugas integrasi API sederhana dan laporan pengujian performa.
\end{asesmen}

\begin{checklist}
  \item Saya dapat melakukan request API menggunakan fetch.
  \item Saya memahami konsep testing aplikasi web.
  \item Saya mengetahui metrik dasar performa web.
\end{checklist}

\begin{rangkuman}
Bab ini membahas integrasi API serta evaluasi dan testing aplikasi web. Kemampuan ini penting untuk membangun aplikasi yang terhubung dan berkualitas.

\textbf{Poin Kunci:}
\begin{itemize}
  \item API memudahkan pertukaran data antara front-end dan back-end.
  \item Testing memastikan fitur berjalan sesuai kebutuhan.
  \item Evaluasi performa membuat aplikasi lebih cepat dan stabil.
\end{itemize}
\end{rangkuman}
