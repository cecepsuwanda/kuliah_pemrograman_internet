\begin{rangkuman}
Bab ini membahas event handling, validasi form, dan error handling sebagai \textbf{Unit Materi Inti} berbasis Sub-CPMK 2.3. Materi fungsi lanjutan dan metode array fungsional (\code{map}, \code{filter}, \code{reduce}) menjadi fondasi. Manipulasi DOM lanjutan dan event handling (addEventListener, propagation, delegation) memungkinkan antarmuka responsif. Validasi form real-time menghubungkan event, DOM, dan logika bisnis. Penanganan error dengan \code{try/catch} memastikan aplikasi tetap stabil.

\textbf{Poin Kunci:}
\begin{itemize}
  \item \textbf{Fungsi lanjutan} (Sub-CPMK 2.3): arrow function, default parameter, rest parameter; \code{map}, \code{filter}, \code{reduce} untuk transformasi array.
  \item \textbf{DOM dan event}: traversing, membuat/menghapus elemen; addEventListener, event bubbling, event delegation untuk antarmuka dinamis.
  \item \textbf{Validasi form real-time}: event \code{input}, \code{change}, \code{blur}; pesan error yang jelas meningkatkan UX; validasi client melengkapi (bukan menggantikan) validasi server.
  \item \textbf{Error handling}: \code{try/catch}, \code{throw} untuk operasi DOM atau JSON yang berisiko gagal.
  \item Event delegation dan DOM manipulation efisien untuk antarmuka dinamis; gunakan \code{try/catch} untuk operasi berisiko.
\end{itemize}
