\section{Selector dan Cara Menyisipkan CSS}
CSS digunakan untuk mengatur tampilan elemen HTML agar lebih menarik dan konsisten. Anda dapat menyisipkan CSS secara inline, internal, atau eksternal, dan praktik terbaik adalah menggunakan file eksternal. Memahami selector adalah kunci untuk memilih elemen yang ingin di-styling \cite{mdnCss}.

Selector dasar meliputi elemen, class, dan id. Selector ini memungkinkan Anda mengatur warna, ukuran teks, dan margin dengan mudah. Dengan pemahaman selector, Anda dapat membuat desain yang terstruktur dan mudah dirawat.

\begin{webcode}[language=CSS, caption={Contoh Selector Dasar CSS}]
h1 {
  color: #2a6fb0;
}

.card {
  padding: 16px;
  border: 1px solid #ddd;
}

#judul-utama {
  font-size: 24px;
}
\end{webcode}
