\section{Tujuan Buku Ajar}

Buku ajar ini dirancang sebagai panduan komprehensif untuk menguasai \textit{Pemrograman Web} dengan memanfaatkan teknologi web modern \cite{mdn-learn}. Fokus utama buku ini adalah pada penguasaan analisis kebutuhan dan perancangan antarmuka (wireframe, mockup, aksesibilitas), arsitektur web, pemrograman client-side (HTML5, CSS3, JavaScript), pemrograman server-side dengan PHP, integrasi basis data MySQL, integrasi API, pengujian, keamanan, dan deployment. Tujuan spesifik buku ini adalah memberikan landasan yang kuat bagi mahasiswa pemula untuk mengembangkan aplikasi web yang fungsional dan aman.

Pembelajaran pemrograman web mencakup berbagai lapisan teknologi yang saling melengkapi. Di sisi client, mahasiswa akan mempelajari struktur konten dengan HTML5, presentasi visual dengan CSS3, dan interaktivitas dengan JavaScript sesuai standar web yang ditetapkan oleh W3C dan WHATWG \cite{whatwg}. Di sisi server, mahasiswa akan diperkenalkan pada pemrosesan request dengan PHP, koneksi basis data MySQL, serta integrasi aplikasi web dinamis menggunakan lingkungan XAMPP \cite{xampp}.

Buku ini bersifat tutorial dan ditujukan untuk pemula yang belum memiliki pengalaman dalam pengembangan web. Setiap konsep dijelaskan langkah demi langkah dengan contoh kode yang dapat dijalankan langsung. Mahasiswa akan banyak berlatih dengan membuat halaman web sederhana, form interaktif, dan aplikasi CRUD dasar sebagai fondasi untuk proyek yang lebih kompleks.

Setelah mempelajari buku ini secara menyeluruh, mahasiswa diharapkan mampu mencapai learning outcomes berikut:

\begin{enumerate}
  \item Memahami arsitektur client-server dan protokol HTTP \cite{mdn-http}
  \item Membangun antarmuka web responsif dengan HTML5, CSS3, dan JavaScript
  \item Mengimplementasikan logika server-side dan integrasi basis data
  \item Mengintegrasikan aplikasi web dengan PHP dan MySQL
  \item Menerapkan prinsip keamanan web dasar (validasi input, prepared statement, session)
\end{enumerate}
