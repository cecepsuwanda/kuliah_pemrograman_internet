\section{Testing dan Evaluasi Performa}
Pengujian memastikan aplikasi berjalan sesuai kebutuhan pengguna. Testing bisa dilakukan secara manual dengan skenario sederhana, atau otomatis menggunakan alat pengujian. Evaluasi performa membantu Anda mengetahui apakah aplikasi lambat atau berat \cite{mdnToolsTesting}.

Metrik dasar seperti waktu muat halaman dan ukuran file bisa diukur melalui DevTools. Anda juga bisa memperkecil ukuran gambar atau meminimalkan JavaScript untuk mempercepat aplikasi. Kebiasaan evaluasi ini membuat aplikasi lebih andal.
