\begin{aktivitas}
  \item \textbf{Praktik:} Implementasi Fetch API: ambil data dari public API, tampilkan di halaman dengan loading dan error state.
  \item \textbf{Praktik:} Tulis class \code{Product} dengan constructor dan method; gunakan \code{extends} untuk pewarisan.
  \item \textbf{Eksplorasi:} Closure (counter privat), RegExp (validasi format), dan Promise/async-await (fetch berurutan).
  \item \textbf{Latihan:} Gunakan Bootstrap atau Tailwind CSS untuk membuat layout responsif dan komponen UI.
  \item \textbf{Eksplorasi:} jQuery atau framework (Vue/React) untuk membuat UI dinamis sederhana; bandingkan kelebihan masing-masing.
  \item \textbf{Diskusi:} Presentasikan proyek SPA sederhana di depan kelompok; terima umpan balik tentang Fetch, keamanan XSS, dan penggunaan library/framework.
\end{aktivitas}

\begin{rangkuman}
Bab ini membahas JavaScript lanjutan serta library/framework front-end untuk UI dinamis sebagai \textbf{Unit Materi Inti} berbasis Sub-CPMK 2.4. Materi Timer, asynchronous (callback, Promise, async/await), dan Fetch API memungkinkan komunikasi dengan server. ES6+ (destructuring, spread, modules), class/pewarisan, closure, dan prototype menjadi fondasi OOP. Library/framework (jQuery, Bootstrap, Vue, React) dan tooling (bundler, linter) mempercepat pengembangan UI dinamis. Keamanan (XSS) dan performa harus diperhatikan sejak awal.

\textbf{Poin Kunci:}
\begin{itemize}
  \item \textbf{Timer dan Async} (Sub-CPMK 2.4): setTimeout, setInterval; callback, Promise, async/await untuk operasi asinkron.
  \item \textbf{Fetch API}: GET/POST request, error handling, loading state; standar modern komunikasi dengan server.
  \item \textbf{ES6+ dan OOP}: destructuring, spread, modules; class, pewarisan, closure, prototype, pola desain (module, singleton).
  \item \textbf{Library dan framework}: jQuery, Bootstrap, Vue, React untuk UI dinamis; perbedaan library vs framework.
  \item \textbf{Keamanan dan performa}: XSS dan pencegahan (gunakan \code{textContent}, sanitasi input); profiling dan praktik baik.
  \item Framework mempercepat development; fondasi JavaScript yang kuat (async, Fetch, ES6+) adalah prasyaratnya.
\end{itemize}

\end{rangkuman}
