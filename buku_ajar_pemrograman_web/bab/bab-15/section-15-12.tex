\section{Session: Memulai, Menyimpan, Regenerasi ID, dan Pengelolaan Aman}

Session menyimpan data pengguna di server dan mengidentifikasi pengguna melalui cookie berisi \textbf{session ID}. Langkah aman: panggil \code{session\_start()} di awal; simpan data di \code{\$\_SESSION}; \textbf{regenerasi ID} setelah login berhasil (\code{session\_regenerate\_id(true)}) untuk mencegah session fixation; atur cookie session dengan atribut keamanan. Saat logout, hapus seluruh data session dan hancurkan cookie \cite{phpManual}.

\begin{webcode}[language=PHP, caption={Pengelolaan session yang aman}]
<?php
// Konfigurasi sebelum session_start()
ini_set('session.cookie_httponly', 1);
ini_set('session.cookie_secure', 1);
ini_set('session.cookie_samesite', 'Strict');
ini_set('session.use_strict_mode', 1);
session_start();

// Login: regenerasi ID untuk cegah fixation
function loginUser(int $userId, string $nama): void {
  session_regenerate_id(true);
  $_SESSION['user_id']    = $userId;
  $_SESSION['nama']       = $nama;
  $_SESSION['login_time'] = time();
}

// Logout: hapus semua data
function logoutUser(): void {
  $_SESSION = [];
  if (ini_get("session.use_cookies")) {
    $p = session_get_cookie_params();
    setcookie(session_name(), '', time() - 42000,
      $p["path"], $p["domain"], $p["secure"], $p["httponly"]);
  }
  session_destroy();
}
?>
\end{webcode}
