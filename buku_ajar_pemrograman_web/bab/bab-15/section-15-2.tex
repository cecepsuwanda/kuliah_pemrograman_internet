\section{Langkah Deployment dan Administrasi Dasar}

Langkah umum deployment: (1) Siapkan server (VPS atau layanan hosting) dengan PHP, MySQL, dan server web; (2) Upload file aplikasi (FTP, SFTP, atau Git); (3) Konfigurasi virtual host dan document root; (4) Set permission file dan folder yang aman; (5) Konfigurasi basis data dan kredensial; (6) Aktifkan HTTPS (SSL/TLS); (7) Uji akses dan fungsionalitas \cite{mdn-learn}, \cite{php-right-way}.

Administrasi dasar meliputi: pemantauan log error, backup basis data dan file, pembaruan keamanan (PHP, MySQL, OS), serta pemantauan uptime \cite{mysql}. Sub-CPMK 4.3 menuntut mahasiswa mampu melakukan deployment ke lingkungan produksi dan administrasi dasar (server/web, basis data).
