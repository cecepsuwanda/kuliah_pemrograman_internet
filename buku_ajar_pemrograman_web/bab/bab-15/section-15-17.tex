\section{Alat Load Testing: JMeter, Locust, atau k6}

Beberapa alat populer untuk load testing: \textbf{Apache JMeter} (GUI-based, Java, mendukung banyak protokol), \textbf{Locust} (Python, menulis skenario sebagai kode Python, cocok untuk developer), dan \textbf{k6} (modern, JavaScript-based, ringan, cocok untuk CI/CD). Pemilihan alat tergantung pada kebutuhan: JMeter untuk skenario kompleks, Locust untuk developer Python, k6 untuk integrasi pipeline modern \cite{mdnToolsTesting}.
