  \item Buat komponen card produk menggunakan box model lengkap: atur \code{padding}, \code{border}, \code{border-radius}, \code{margin: auto} untuk centering, dan gunakan \code{box-sizing: border-box}.

\begin{asesmen}
Asesmen berupa proyek styling halaman web lengkap yang menggabungkan seluruh konsep Bab 6:
\begin{enumerate}
  \item Buat halaman produk dengan: CSS eksternal, selector elemen/class/id, skema warna konsisten, tipografi menggunakan web font, dan background gradient.
  \item Halaman harus memiliki: card produk yang menggunakan box model (\code{border-box}), menu navigasi horizontal dari list, tabel harga dengan zebra striping, dan tautan dengan styling state lengkap.
  \item Gunakan satuan \code{rem} untuk font-size dan spacing, \code{\%} untuk layout, dan \code{calc()} minimal satu kasus.
  \item Fokus indikator (\code{:focus} / \code{:focus-visible}) harus terlihat pada semua elemen interaktif.
\end{enumerate}
Penilaian: ketepatan selector dan specificity (20\%), box model dan layout (25\%), warna dan tipografi (20\%), aksesibilitas dan kode bersih (20\%), konsistensi satuan (15\%).
\end{asesmen}

  \item Saya menggunakan \code{box-sizing: border-box} secara global dan memahami perbedaan dari \code{content-box}.

\begin{rangkuman}
Bab ini mengenalkan CSS sebagai bahasa pengatur tampilan halaman web, dimulai dari konsep cascading dan specificity yang menentukan aturan mana yang diterapkan saat terjadi konflik. Tiga metode penyisipan CSS---inline, internal, dan eksternal---dibahas beserta kelebihan dan kekurangan masing-masing, dengan CSS eksternal sebagai praktik terbaik.

Sintaks CSS dibangun dari selector, properti, dan nilai. Selector dasar (elemen, class, id) serta selector gabungan (descendant, child, grouping) memungkinkan targeting yang presisi. Warna dan background mencakup beragam format (hex, rgb, hsl, gradient), sementara tipografi dikontrol melalui font-family, font-size, line-height, dan web fonts.

Box model---terdiri dari content, padding, border, dan margin---adalah fondasi layout CSS. Properti \code{box-sizing: border-box} menyederhanakan perhitungan dimensi. Properti \code{display} mengontrol perilaku elemen dalam flow layout, dengan tiga mode utama: block, inline, dan inline-block. Styling tautan (pseudo-class LVHA) dan aksesibilitas fokus keyboard menjadi perhatian penting. Terakhir, pemilihan satuan yang tepat---rem untuk font-size, em/rem untuk spacing, \% untuk layout---memastikan styling yang skalabel dan mudah dirawat.

\textbf{Poin Kunci:}
\begin{itemize}
  \item Cascading dan specificity menentukan aturan CSS mana yang berlaku; pahami untuk menghindari konflik.
  \item Box model (content, padding, border, margin) mengontrol spasi dan ukuran; gunakan \code{border-box} secara global.
  \item Selector class direkomendasikan untuk styling komponen modular; id untuk anchor dan JavaScript.
  \item Satuan rem direkomendasikan untuk font-size demi aksesibilitas dan skalabilitas.
  \item Selalu sediakan indikator fokus yang jelas pada elemen interaktif.
\end{itemize}
\end{rangkuman}
