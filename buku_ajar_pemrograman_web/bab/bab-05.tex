\begin{aktivitas}
  \item \textbf{Praktik:} Buat folder proyek dan file \code{index.html} dengan struktur HTML5 lengkap (doctype, head dengan meta charset dan viewport, body).
  \item \textbf{Praktik:} Tambahkan header dengan judul h1, navigasi dengan unordered list, dan main content dengan minimal 3 section.
  \item \textbf{Praktik:} Buat halaman \code{tentang.html} yang menggunakan elemen semantik (article, aside, footer) dan link internal dari index.html.
  \item \textbf{Praktik:} Sisipkan gambar dengan figure dan figcaption, pastikan atribut alt deskriptif dan dimensi width/height tersedia.
  \item \textbf{Praktik:} Buat link eksternal ke 3 website berbeda dengan atribut \code{rel="noopener noreferrer"} dan \code{target="\_blank"}.
  \item \textbf{Latihan:} Gunakan DevTools browser untuk inspeksi struktur DOM dan verifikasi hierarki heading.
\end{aktivitas}

\begin{latihan}
  \item Jelaskan perbedaan tag, elemen, dan atribut dalam HTML. Berikan masing-masing 2 contoh konkret.
  \item Buat diagram hierarki heading yang benar untuk halaman artikel berita dengan struktur: judul utama, 2 subbagian masing-masing dengan 2 sub-subbagian.
  \item Identifikasi 5 kesalahan struktural dalam kode HTML berikut dan jelaskan cara memperbaikinya (termasuk nesting yang salah dan elemen semantik yang hilang).
  \item Buat tabel perbandingan antara \code{<em>} vs \code{<i>} dan \code{<strong>} vs \code{<b>}, jelaskan kapan menggunakan masing-masing.
  \item Jelaskan mengapa atribut \code{alt} pada gambar penting untuk aksesibilitas dan berikan contoh alt text yang baik dan buruk untuk gambar yang sama.
  \item Buat daftar berurutan (ol) untuk resep masakan dengan langkah-langkah yang jelas dan nested list untuk sub-langkah persiapan.
  \item Buat description list (dl) untuk glosarium 5 istilah teknis web development dengan definisi yang komprehensif.
  \item Desain struktur semantik HTML5 untuk halaman landing produk dengan: header, hero section dengan gambar, fitur produk (3 card), testimonial, dan footer dengan kontak.

Asesmen Bab 5 merupakan \textbf{Evaluasi Kinerja} yang mengukur Sub-CPMK 2.1. Terkait RPS: Praktikum 1 (Pertemuan 4). Komponen penilaian meliputi:

\begin{itemize}[leftmargin=*]
  \item \textbf{Struktur Dokumen (30\%):} Keberadaan doctype, html, head lengkap dengan meta wajib (charset, viewport, title), dan body. Mengukur Sub-CPMK 2.1.
  \item \textbf{Elemen Semantik (25\%):} Penggunaan header, nav, main, article/section, aside, footer sesuai konteks. Mengukur Sub-CPMK 2.1.
  \item \textbf{Konten dan Format (20\%):} Hierarki heading yang benar, paragraf, daftar, dan elemen teks yang tepat. Mengukur Sub-CPMK 2.1.
  \item \textbf{Link dan Gambar (15\%):} Atribut href yang valid, keamanan link eksternal (noopener noreferrer), atribut alt dan dimensi pada gambar. Mengukur Sub-CPMK 2.1.
  \item \textbf{Validasi dan Best Practice (10\%):} Passing W3C Validator, kode rapi dengan indentasi konsisten. Mengukur Sub-CPMK 2.1.

\begin{rangkuman}
Bab ini membahas HTML dasar, struktur dokumen, dan elemen semantik sebagai \textbf{Unit Materi Inti} berbasis Sub-CPMK 2.1.

\textbf{Poin Kunci:}
\begin{itemize}
  \item \textbf{HTML} (Sub-CPMK 2.1): Bahasa markup deklaratif untuk struktur dan makna konten; struktur standar terdiri dari \code{head} (metadata) dan \code{body} (konten terlihat).
  \item \textbf{Sintaks dasar}: Tag, elemen, atribut; doctype HTML5; hierarki heading yang benar; elemen semantik (em, strong, mark) vs presentasional (i, b).
  \item \textbf{Link dan gambar}: href, link internal/eksternal/anchor; keamanan \code{rel="noopener noreferrer"}; alt untuk aksesibilitas, width/height untuk layout stability.
  \item \textbf{Elemen semantik HTML5}: header, nav, main, article, section, aside, footer---meningkatkan aksesibilitas, SEO, dan maintainability kode.
  \item \textbf{Aksesibilitas}: Alt text untuk gambar, hierarki heading logis, elemen semantik untuk screen readers.
\end{itemize}

\end{rangkuman}
