\section{Library dan Framework Front-End}
Library seperti jQuery menyederhanakan manipulasi DOM, sedangkan framework seperti Vue atau React membantu membangun UI kompleks. Penggunaan library memungkinkan Anda membuat komponen dinamis dengan lebih cepat. Untuk pemula, memahami konsep dasar library cukup untuk meningkatkan produktivitas \cite{w3schoolsTutorials}.

Contoh sederhana penggunaan jQuery adalah memilih elemen dan menambahkan event tanpa banyak kode. Framework modern menekankan konsep komponen agar UI mudah dipelihara. Dalam praktik, Anda dapat memilih library sesuai kebutuhan proyek dan skala aplikasi.

\begin{webcode}[language=JavaScript, caption={Contoh jQuery Sederhana}]
$("#tombol").on("click", function () {
  $("#pesan").text("Halo dari jQuery!");
});
\end{webcode}
