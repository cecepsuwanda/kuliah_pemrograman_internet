  \item Jelaskan perbedaan \code{var}, \code{let}, dan \code{const} dalam hal scope, hoisting, dan re-assignment.
  \item Apa yang dimaksud dengan type coercion? Berikan contoh berbahaya dan cara mengatasinya dengan \code{===}.
  \item Jelaskan perbedaan function declaration, function expression, dan arrow function. Kapan menggunakan masing-masing?
  \item Apa itu closure? Berikan contoh sederhana dan jelaskan mengapa closure berguna.
  \item Bandingkan \code{textContent} dan \code{innerHTML}. Mengapa \code{innerHTML} berisiko untuk input pengguna?
  \item Jelaskan event bubbling dan event delegation. Berikan contoh penggunaan event delegation pada daftar item.
  \item Buat fungsi validasi form yang memeriksa nama (tidak kosong), email (mengandung ``@''), dan password (minimal 8 karakter).
  \item Buat counter interaktif dengan tombol + dan - yang mengubah angka di halaman menggunakan DOM.
\end{latihan}

\begin{asesmen}
Asesmen berupa proyek aplikasi web interaktif yang menggabungkan seluruh konsep Bab 8:
\begin{enumerate}
  \item Buat halaman pendaftaran dengan form yang memiliki validasi JavaScript: nama, email, password, dan konfirmasi password.
  \item Validasi harus bersifat real-time (event \code{input}) dan saat submit (event \code{submit} + \code{preventDefault()}).
  \item Tampilkan pesan error di bawah setiap field menggunakan \code{textContent} (bukan \code{innerHTML}).
  \item Gunakan array objek untuk menyimpan data pendaftar; tampilkan dalam tabel HTML yang dibangun secara dinamis dengan DOM API.
  \item Tambahkan fitur toggle dark mode menggunakan \code{classList.toggle()}.
\end{enumerate}
Penilaian: validasi form (25\%), manipulasi DOM (25\%), JavaScript dasar (20\%), keamanan (15\%), kode bersih (15\%).
\end{asesmen}

\begin{rangkuman}
Bab ini membahas JavaScript dasar dan pengenalan DOM. Materi dimulai dari pengenalan JavaScript dan ekosistemnya, cara menjalankan kode di browser (konsol, script tag, file eksternal), dan sintaks dasar (statement, komentar, strict mode).

Fondasi bahasa meliputi variabel (\code{let}/\code{const} sebagai standar modern, \code{var} sebagai legacy), lima tipe data primitif, operator (dengan penekanan pada \code{===} untuk perbandingan aman), dan konversi tipe. String dibahas dengan metode umum dan template literal ES6.

Control flow mencakup percabangan (\code{if/else}, \code{switch}, ternary) dan perulangan (\code{for}, \code{while}, \code{for...of}). Fungsi sebagai first-class citizens dibahas dalam tiga bentuk (declaration, expression, arrow function) beserta default parameter. Scope (global, function, block) dan closure melengkapi pemahaman tentang visibilitas variabel.

Struktur data meliputi array (dengan metode fungsional: \code{map}, \code{filter}, \code{reduce}) dan objek literal (properti, metode, destructuring). DOM diperkenalkan sebagai pohon objek yang merepresentasikan halaman; akses elemen, manipulasi konten (\code{textContent} vs \code{innerHTML} dengan pertimbangan keamanan), dan event handling (\code{addEventListener}, event bubbling, \code{preventDefault()}) melengkapi kemampuan membangun halaman web interaktif.

\textbf{Poin Kunci:}
\begin{itemize}
  \item Gunakan \code{const} sebagai default; \code{let} jika perlu re-assign; hindari \code{var}.
  \item Selalu gunakan \code{===} untuk perbandingan; waspadai type coercion.
  \item Fungsi adalah first-class citizens; arrow function untuk callback, function declaration untuk method.
  \item DOM memungkinkan JavaScript memanipulasi halaman; gunakan \code{textContent} (bukan \code{innerHTML}) untuk keamanan.
  \item Event handling dengan \code{addEventListener}; event delegation untuk efisiensi.
\end{itemize}
\end{rangkuman}
