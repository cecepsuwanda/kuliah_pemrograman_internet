\begin{aktivitas}
  \item \textbf{Praktik:} Buat halaman dengan positioning: fixed header, sticky sidebar, dan elemen absolute (tooltip atau badge).
  \item \textbf{Praktik:} Buat navbar horizontal dengan Flexbox: logo di kiri, menu di kanan, responsif menjadi vertikal di mobile.
  \item \textbf{Praktik:} Buat grid kartu produk responsif menggunakan \code{repeat(auto-fill, minmax(280px, 1fr))}; tambahkan halaman layout dengan \code{grid-template-areas}.
  \item \textbf{Praktik:} Terapkan pendekatan mobile-first dengan media queries: base style untuk mobile, lalu \code{min-width} breakpoint untuk tablet dan desktop.
  \item \textbf{Praktik:} Buat form yang di-styling konsisten (\code{font-family: inherit}) dengan indikator fokus jelas dan dropdown menu yang aksesibel.
  \item \textbf{Proyek mini:} Buat halaman portfolio responsif yang menggabungkan Flexbox, Grid, media queries, transition, animasi, dan custom properties; implementasikan dark mode.
  \item \textbf{Studi kasus:} Analisis satu situs web responsif: identifikasi penggunaan Flexbox vs Grid, breakpoint media query, dan pendekatan mobile-first; bandingkan dengan praktik terbaik.
\end{aktivitas}

\begin{latihan}
  \item Jelaskan perbedaan \code{position: relative}, \code{absolute}, \code{fixed}, dan \code{sticky}. Kapan menggunakan masing-masing?
  \item Apa perbedaan Flexbox dan CSS Grid? Berikan contoh komponen yang cocok untuk masing-masing.
  \item Jelaskan cara kerja \code{repeat(auto-fill, minmax(250px, 1fr))} dan mengapa grid ini responsif tanpa media query.
  \item Apa keunggulan pendekatan mobile-first dibanding desktop-first? Jelaskan dengan contoh media query.
  \item Jelaskan perbedaan \code{transition} dan \code{animation} dalam CSS. Kapan menggunakan masing-masing?
  \item Apa itu BEM? Berikan contoh penamaan Block, Element, dan Modifier untuk komponen card.
  \item Mengapa animasi dengan \code{transform}/\code{opacity} lebih performan dari animasi dengan \code{width}/\code{margin}?
  \item Jelaskan bagaimana \code{@media (prefers-reduced-motion: reduce)} meningkatkan aksesibilitas.
  \item \textbf{Refleksi:} Dari materi bab ini, indikator Sub-CPMK 2.2 mana yang sudah Anda kuasai? Area mana yang masih perlu Anda tingkatkan untuk mencapai kompetensi layout dan desain responsif?
\end{latihan}

\begin{rangkuman}
Bab ini membahas Flexbox, Grid, dan desain responsif sebagai \textbf{Unit Materi Inti} berbasis Sub-CPMK 2.2. Materi meliputi seluruh spektrum CSS layout dan responsif, dimulai dari normal flow dan positioning (static, relative, absolute, fixed, sticky) hingga sistem layout modern: Flexbox untuk komponen satu dimensi dan CSS Grid untuk layout dua dimensi. Float dibahas sebagai metode legacy yang tetap relevan untuk memahami kode lama.

Media queries dan pendekatan mobile-first menjadi fondasi desain responsif, dilengkapi unit viewport (vw, vh, dvh) dan teknik fluid typography dengan \code{clamp()}. Styling form dan navigasi (termasuk dropdown) dibahas dengan perhatian pada konsistensi dan aksesibilitas.

Transition dan animation memberikan interaksi yang halus; transform memungkinkan manipulasi visual tanpa mempengaruhi layout; ketiganya dioptimalkan oleh GPU untuk performa terbaik. CSS custom properties menciptakan design system yang konsisten dan mendukung fitur seperti dark mode.

Spesifisitas, cascade, dan CSS Layers mengontrol prioritas aturan. Pseudo-class dan pseudo-element lanjutan memperluas kemampuan selector. Aksesibilitas, organisasi kode (BEM), dan best practices performa melengkapi penguasaan Sub-CPMK 2.2.

\textbf{Poin Kunci:}
\begin{itemize}
  \item Flexbox (1D) dan Grid (2D) adalah fondasi layout modern; keduanya saling melengkapi.
  \item Mobile-first (\code{min-width}) menghasilkan CSS lebih ringan dan direkomendasikan.
  \item Animasikan hanya \code{transform}/\code{opacity} untuk performa optimal.
  \item Custom properties menciptakan design system yang konsisten dan mendukung dark mode.
  \item Aksesibilitas bukan opsional: indikator fokus, reduced motion, dan kontras warna wajib diperhatikan.
\end{itemize}
\end{rangkuman}
