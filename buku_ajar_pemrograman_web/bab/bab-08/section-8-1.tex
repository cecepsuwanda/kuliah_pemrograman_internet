\section{Sintaks Dasar dan Struktur Program}
JavaScript adalah bahasa pemrograman yang membuat halaman web menjadi interaktif. Anda dapat menulis variabel, percabangan, dan perulangan untuk mengontrol alur program. Konsep dasar ini menjadi fondasi sebelum berinteraksi dengan DOM \cite{jsInfo}.

Salah satu cara belajar efektif adalah mencoba kode kecil secara bertahap di console browser. Anda bisa melihat langsung hasilnya tanpa harus membuat file terpisah. Kebiasaan ini membantu Anda memahami perilaku kode dengan cepat.

\begin{webcode}[language=JavaScript, caption={Contoh Variabel dan Percabangan}]
const nilai = 75;
if (nilai >= 70) {
  console.log("Lulus");
} else {
  console.log("Belum lulus");
}
\end{webcode}
