\section{Pengenalan Pemrograman Server-Side dan PHP}

Pemrograman server-side menangani logika di server: pemrosesan form, akses basis data, dan pembuatan respons dinamis \cite{mdn-learn}. PHP adalah bahasa server-side yang dijalankan oleh web server dan terintegrasi dengan HTML \cite{php-manual}. Di lingkungan XAMPP, PHP memproses request, berinteraksi dengan MySQL, dan mengembalikan HTML atau JSON \cite{xampp}. Variabel \texttt{\$\_GET} dan \texttt{\$\_POST} menyimpan data dari request; \texttt{\$\_SERVER} berisi informasi server dan metode request \cite{w3schools-php}.
