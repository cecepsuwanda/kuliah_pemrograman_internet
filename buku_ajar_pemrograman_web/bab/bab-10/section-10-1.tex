\section{Sintaks Dasar PHP dan Output}
PHP adalah bahasa pemrograman server-side yang berjalan di server dan menghasilkan output HTML. Anda dapat menulis variabel, percabangan, dan perulangan untuk memproses data. Konsep dasar ini membuat halaman menjadi dinamis \cite{phpManual}.

Kode PHP ditulis di dalam tag \code{<?php ... ?>} dan bisa disisipkan ke dalam file HTML. Saat server memproses file, PHP akan menghasilkan output yang dikirim ke browser. Dengan memahami alur ini, Anda bisa membangun halaman yang berubah sesuai data.

\begin{webcode}[language=PHP, caption={Contoh Output PHP}]
<?php
$nama = "Alya";
echo "Halo, " . $nama;
?>
\end{webcode}
