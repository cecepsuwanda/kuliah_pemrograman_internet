\begin{checklist}

\begin{rangkuman}
Bab ini membahas Flexbox dan Grid sebagai teknik layout modern serta konsep responsif. Penguasaan keduanya memudahkan Anda membuat tampilan yang rapi di berbagai perangkat.

\textbf{Poin Kunci:}
\begin{itemize}
  \item Flexbox cocok untuk layout satu dimensi.
  \item Grid cocok untuk layout dua dimensi.
  \item Media query membuat desain adaptif.
\end{itemize}
\end{rangkuman}
