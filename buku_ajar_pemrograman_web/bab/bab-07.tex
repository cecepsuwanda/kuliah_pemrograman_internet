\begin{aktivitas}
  \item \textbf{Praktik:} Buat file CSS eksternal dan hubungkan ke halaman HTML; terapkan selector elemen, class, dan id untuk mengatur warna, tipografi, dan spasi.
  \item \textbf{Praktik:} Buat komponen card produk menggunakan box model lengkap: atur \code{padding}, \code{border}, \code{border-radius}, \code{margin: auto} untuk centering, dan gunakan \code{box-sizing: border-box}.
  \item \textbf{Praktik:} Buat menu navigasi horizontal dari \code{<ul>} dengan menghilangkan bullet, menggunakan \code{display: inline-block} atau Flexbox, dan styling state tautan (\code{:link}, \code{:visited}, \code{:hover}, \code{:focus}).
  \item \textbf{Praktik:} Styling tabel data HTML: gunakan \code{border-collapse}, \code{padding} pada \code{th}/\code{td}, background header, dan zebra striping dengan \code{:nth-child(even)}.

\begin{rangkuman}
Bab ini membahas CSS dasar, selector, box model, dan tipografi sebagai \textbf{Unit Materi Inti} berbasis Sub-CPMK 2.2. CSS merupakan bahasa pengatur tampilan halaman web, dimulai dari konsep cascading dan specificity yang menentukan aturan mana yang diterapkan saat terjadi konflik. Tiga metode penyisipan CSS---inline, internal, dan eksternal---dibahas beserta kelebihan dan kekurangan masing-masing, dengan CSS eksternal sebagai praktik terbaik.

Sintaks CSS dibangun dari selector, properti, dan nilai. Selector dasar (elemen, class, id) serta selector gabungan (descendant, child, grouping) memungkinkan targeting yang presisi. Warna dan background mencakup beragam format (hex, rgb, hsl, gradient), sementara tipografi dikontrol melalui font-family, font-size, line-height, dan web fonts.

Box model---terdiri dari content, padding, border, dan margin---adalah fondasi layout CSS. Properti \code{box-sizing: border-box} menyederhanakan perhitungan dimensi. Properti \code{display} mengontrol perilaku elemen dalam flow layout, dengan tiga mode utama: block, inline, dan inline-block. Styling tautan (pseudo-class LVHA) dan aksesibilitas fokus keyboard menjadi perhatian penting. Terakhir, pemilihan satuan yang tepat---rem untuk font-size, em/rem untuk spacing, \% untuk layout---memastikan styling yang skalabel dan mudah dirawat.

\textbf{Poin Kunci:}
\begin{itemize}
  \item Cascading dan specificity menentukan aturan CSS mana yang berlaku; pahami untuk menghindari konflik.
  \item Box model (content, padding, border, margin) mengontrol spasi dan ukuran; gunakan \code{border-box} secara global.
  \item Selector class direkomendasikan untuk styling komponen modular; id untuk anchor dan JavaScript.
  \item Satuan rem direkomendasikan untuk font-size demi aksesibilitas dan skalabilitas.
  \item Selalu sediakan indikator fokus yang jelas pada elemen interaktif.
\end{itemize}
\end{rangkuman}
