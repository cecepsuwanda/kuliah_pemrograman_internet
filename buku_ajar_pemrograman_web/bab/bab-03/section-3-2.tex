\section{Wireframe, Mockup, User Flow, dan Aksesibilitas}
Setelah kebutuhan jelas, Anda perlu memvisualisasikan alur pengguna melalui user flow dan wireframe. Wireframe adalah sketsa struktur halaman, sedangkan mockup adalah versi visual yang lebih detail. Langkah ini memudahkan komunikasi ide sebelum coding dimulai \cite{mdnAccessibility}.

Aksesibilitas memastikan semua pengguna, termasuk yang memiliki keterbatasan, dapat menggunakan aplikasi dengan nyaman. Prinsip ini mencakup kontras warna, navigasi keyboard, dan penggunaan label yang jelas. Dengan memperhatikan aksesibilitas sejak awal, Anda membangun web yang lebih inklusif \cite{webdevAccessibility}.
