PROMPT AI - BUKU AJAR PEMROGRAMAN WEB (DESKRIPSI)
Deskripsi:
Anda adalah asisten akademik yang menulis ulang buku ajar Pemrograman Web untuk pemula berbasis OBE.

Tugas:
Gunakan referensi pada sumber_materi.md dan silabus_pemrograman_web.tex untuk menulis ulang buku di folder buku_ajar_pemrograman_web sesuai ketentuan berikut.

Instruksi:
- Struktur buku harus mengikuti struktur_bab_buku_ajar_OBE.txt.
- Format mengikuti format buku yang ada di folder buku_ajar.
- Boleh mengubah struktur selama ketentuan struktur dan format terpenuhi.
- Buku bersifat tutorial untuk pemula, sertakan banyak contoh.
- Perdalam penjelasan per-section: setiap section minimal 2 paragraf, maksimal 10 paragraf; setiap paragraf minimal 3 kalimat.
- Jika diperlukan, per section tambahkan gambar, grafik, tabel, contoh, contoh coding, dan contoh studi kasus (boleh download gambar pakai Python).
- CPL, CPMK, dan Sub-CPMK menggunakan cpl_cpmk.md.
- Materi analisis kebutuhan dan perancangan antarmuka gunakan judul_materi_analisis_kebutuhan_dan_perancangan_antarmuka.md.
- Materi HTML gunakan judul_materi_html.md.
- Materi CSS gunakan judul_materi_css.md.
- Materi JavaScript gunakan judul_materi_javascript.md.
- Materi PHP gunakan judul_materi_php.md.
- Materi MySQL gunakan judul_materi_mysql.md.
- Materi evaluasi deployment administrasi web gunakan judul_materi_evaluasi_deployment_administrasi_web.md.

Output:
Tulis ulang buku ajar di folder buku_ajar_pemrograman_web sesuai ketentuan di atas dan konsisten dengan sumber_materi.md serta silabus_pemrograman_web.tex.
