# Judul Materi Belajar MySQL: Dasar sampai Mahir

Daftar judul materi ini disusun berdasarkan referensi pada bagian **"Tutorial MySQL untuk Pemula"** di [sumber_materi.md](sumber_materi.md), serta kurikulum standar dari MySQL Documentation, W3Schools SQL Tutorial, dan Guru99 MySQL Tutorial. Urutan dari dasar sampai mahir mengikuti alur: pengenalan database & SQL → instalasi MySQL → DDL & DML dasar → query lanjut & agregasi → JOIN & subquery → normalisasi & constraint → koneksi PHP (MySQLi) → topik lanjut (view, stored procedure, transaksi, keamanan).

**Pemetaan ke [cpl_cpmk.md](cpl_cpmk.md):** Materi ini mendukung **Sub-CPMK 3.2** (mengelola basis data, operasi CRUD dengan MySQL, koneksi dari PHP). Semua judul sesuai dengan CPMK3; tidak ada yang ditandai tidak sesuai.

---

## Tingkat Dasar

1. Pengenalan database dan SQL
2. Instalasi MySQL dan lingkungan (XAMPP/standalone)
3. Mengakses MySQL: command line dan alat GUI
4. CREATE DATABASE dan USE
5. CREATE TABLE dan tipe data umum (INT, VARCHAR, DATE, TEXT, dll.)
6. Mengubah struktur: ALTER TABLE
7. SELECT dasar dan WHERE
8. Operator perbandingan dan logika dalam WHERE
9. ORDER BY dan LIMIT
10. INSERT, UPDATE, DELETE
11. Fungsi agregat: COUNT, SUM, AVG, MIN, MAX
12. GROUP BY dan HAVING
13. DISTINCT dan alias (kolom serta tabel)
14. Nilai NULL dan penanganannya

---

## Tingkat Menengah

1. Relasi tabel dan konsep kunci (primary key, foreign key)
2. INNER JOIN, LEFT JOIN, RIGHT JOIN
3. Subquery (dasar)
4. PRIMARY KEY dan UNIQUE
5. FOREIGN KEY dan referential integrity
6. Normalisasi basis data: 1NF, 2NF, 3NF
7. Index: pengenalan dan penggunaan
8. Backup dan restore (mysqldump dasar)
9. Pengenalan MySQLi (PHP): koneksi dan menjalankan query
10. Mengambil hasil query (fetch) dengan MySQLi
11. CRUD aplikasi web dengan PHP dan MySQL
12. Prepared statement dengan MySQLi (mencegah SQL injection)

---

## Tingkat Lanjut / Mahir

1. VIEW: membuat dan menggunakan
2. Stored procedure dan stored function (dasar)
3. Trigger (dasar)
4. Transaksi: BEGIN, COMMIT, ROLLBACK
5. Optimasi query: EXPLAIN dan indeks
6. Keamanan: user, hak akses, dan praktik aman
7. Integrasi lanjut MySQL dengan PHP/API
8. Praktik: merancang dan mengelola basis data untuk aplikasi web
