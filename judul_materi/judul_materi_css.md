# Judul Materi Belajar CSS: Dasar sampai Mahir

Daftar judul materi ini disusun berdasarkan referensi pada bagian **"Tutorial CSS untuk Pemula"** di [sumber_materi.md](sumber_materi.md), serta kurikulum standar dari MDN Learn CSS, web.dev Learn CSS, dan W3Schools CSS Tutorial. Urutan dari dasar sampai mahir mengikuti alur: pengenalan → sintaks & selector → warna & teks → box model & layout dasar → Flexbox & Grid → responsif → animasi & transform → topik lanjut.

**Pemetaan ke [cpl_cpmk.md](cpl_cpmk.md):** Materi ini mendukung **Sub-CPMK 2.2** (styling, layout Flexbox/Grid, desain responsif). Semua judul sesuai dengan CPMK2; tidak ada yang ditandai tidak sesuai.

---

## Tingkat Dasar

1. Pengenalan CSS dan perannya dalam styling halaman web
2. Cara menyisipkan CSS: inline, internal, dan eksternal
3. Sintaks dasar CSS: selector, properti, dan nilai
4. Selector dasar: elemen, class, dan id
5. Warna dan background (warna teks, latar belakang, gradient dasar)
6. Teks dan tipografi (font, size, weight, alignment, line-height)
7. Box model: margin, padding, dan border
8. Lebar, tinggi, dan box-sizing
9. Display: block, inline, dan inline-block
10. Link dan styling tautan (state: hover, visited, active)
11. Daftar (list) dan tabel: styling dasar
12. Satuan dan nilai (px, em, rem, %, warna)

---

## Tingkat Menengah

1. Normal flow dan positioning (static, relative, absolute, fixed, sticky)
2. Flexbox: konsep container dan item, arah, alignment, wrap
3. Flexbox: praktik layout dengan flex
4. CSS Grid: grid container, track, dan area
5. CSS Grid: praktik layout dengan grid
6. Float dan clear (konteks legacy)
7. Overflow dan pengelolaan konten yang meluber
8. Media queries dan desain responsif
9. Viewport dan unit responsif (vw, vh, min/max-width)
10. Pendekatan mobile-first dan breakpoint
11. Form: styling input, button, dan form controls
12. Navigasi dan menu (navbar dasar, dropdown konsep)

---

## Tingkat Lanjut / Mahir

1. Transisi (transition): properti, duration, timing function
2. Animasi (animation) dan keyframes
3. Transform: translate, scale, rotate, skew
4. CSS custom properties (variabel)
5. Spesifisitas dan cascade (review dan praktik lanjut)
6. Pseudo-class dan pseudo-element lanjutan
7. Aksesibilitas dalam CSS (focus, reduced motion, kontras)
8. Organisasi kode dan metodologi (BEM, naming)
9. Best practices dan performa CSS
10. Referensi standar: W3C CSS Specifications dan MDN CSS
