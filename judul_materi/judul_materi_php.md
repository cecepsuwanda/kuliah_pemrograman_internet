# Judul Materi Belajar PHP: Dasar sampai Mahir

Daftar judul materi ini disusun berdasarkan referensi pada bagian **"Tutorial PHP untuk Pemula"** di [sumber_materi.md](sumber_materi.md), serta kurikulum standar dari PHP Manual, W3Schools PHP Tutorial, PHP The Right Way, dan MDN Server-side. Urutan dari dasar sampai mahir mengikuti alur: pengenalan & instalasi → sintaks & variabel → tipe data & operator → kontrol alur → fungsi & array → form & superglobal → file & include → OOP → session & cookie → database → keamanan & praktik baik → topik lanjut.

**Pemetaan ke [cpl_cpmk.md](cpl_cpmk.md):** Materi ini mendukung **Sub-CPMK 3.1** (pemrosesan form, struktur dasar, OOP), **Sub-CPMK 3.3** (integrasi API & front–back-end; judul eksplisit ditambah di Tingkat Lanjut), dan **Sub-CPMK 4.2** (keamanan web: validasi, prepared statement, XSS, CSRF). Semua judul sesuai dengan CPMK3 dan CPMK4.

---

## Tingkat Dasar

1. Pengenalan PHP dan perannya dalam pemrograman server-side web
2. Memulai: instalasi PHP dan environment (XAMPP, server lokal)
3. Sintaks dasar PHP: tag, komentar, dan aturan penulisan
4. Variabel: penamaan, tipe dinamis, dan scope dasar
5. Tipe data: string, integer, float, boolean, null, dan tipe lain
6. Operator: aritmetika, perbandingan, logika, penugasan, dan string
7. Echo dan print: menampilkan output
8. String: literal, escape, konkatenasi, dan fungsi string dasar
9. Percabangan: if, else if, else, dan switch
10. Perulangan: for, while, do-while, break, continue
11. Fungsi: deklarasi, parameter, return, dan nilai default
12. Array: array terindeks dan asosiatif, fungsi array dasar
13. Superglobal: $_GET, $_POST, dan $_SERVER (pengenalan)
14. Form HTML dan pemrosesan form dengan PHP
15. Include dan require: mengorganisasi kode dalam banyak file

---

## Tingkat Menengah

1. Pengenalan pemrograman berorientasi objek (OOP) di PHP
2. Class dan object: properti dan metode
3. Constructor dan destructor
4. Pewarisan (inheritance) dan overriding
5. Visibility: public, protected, private
6. Static properti dan metode
7. Session: memulai, menyimpan, dan menghapus session
8. Cookie: membuat dan membaca cookie
9. Unggah file (file upload) dan validasi
10. Penanganan error: error reporting dan penanganan dasar
11. Koneksi database: pengenalan MySQL dan MySQLi
12. Query dasar: SELECT, INSERT, UPDATE, DELETE
13. Form lanjutan: validasi input dan sanitasi dasar
14. Praktik: membangun aplikasi web sederhana (CRUD dasar)

---

## Tingkat Lanjut / Mahir

1. PDO (PHP Data Objects): koneksi dan prepared statement
2. Prepared statement dengan MySQLi: mencegah SQL injection
3. Keamanan web: validasi input, sanitasi, dan praktik aman
4. Keamanan: mitigasi XSS dan CSRF (konsep dan praktik dasar)
5. Exception dan try-catch (penanganan exception)
6. Namespace dan autoloading
7. Pengenalan Composer dan dependency management
8. Praktik terbaik: struktur proyek dan standar penulisan (mengacu PHP The Right Way)
9. Penggunaan PHP dari command line (CLI)
10. RESTful API dasar dengan PHP
11. **Integrasi API dan menghubungkan front-end dengan back-end (request/response, data terstruktur mis. JSON)** *(mendukung Sub-CPMK 3.3)*
12. Pengujian dasar (konsep unit test dan alat)
13. Deployment: menyiapkan aplikasi PHP untuk produksi
